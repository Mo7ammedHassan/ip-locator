# مكتشف المواقع الجغرافية - IP & GPS Locator

مشروع متكامل لتحديد الموقع الجغرافي للمستخدم باستخدام عنوان الـ IP أو إحداثيات الـ GPS الفعالة بدقة 100%.

## 🚀 تشغيل المشروع بأسهل طريقة (ويندوز)

لقد تم إعداد سكربتات تشغيل تلقائية لتسهيل تشغيل المشروع بضغطة زر واحدة دون الحاجة لكتابة أوامر في الترمينال:

1. **تثبيت المكتبات (لأول مرة فقط)**:
   - قم بفتح مجلد المشروع واضغط مرتين على ملف **`setup.bat`**.
   - سيقوم الملف بتثبيت كل المكتبات البرمجية المطلوبة للواجهة والخلفية تلقائياً.

2. **تشغيل المشروع**:
   - اضغط مرتين على ملف **`run.bat`**.
   - سيقوم السيرفر والواجهة بالعمل معاً تلقائياً، وسيفتح متصفحك على الرابط التالي: **[http://localhost:5173](http://localhost:5173)**.

*(ملاحظة: تأكد من أن قاعدة بيانات MongoDB تعمل على جهازك على المنفذ الافتراضي 27017)*

---

## 🛠️ التشغيل اليدوي عبر الترمينال (Terminal)

إذا أردت تشغيل المشروع يدوياً، افتح موجه الأوامر في مجلد المشروع الرئيسي ونفذ التالي:

```bash
# 1. تثبيت كافة المكتبات للمشروع بالكامل
npm run setup

# 2. تشغيل السيرفر والواجهة معاً في نفس الوقت
npm run dev
```

---

## 📂 هيكلية المجلدات (Project Structure)
- **`backend/`**: سيرفر الـ Express مع Mongoose للاتصال بقاعدة البيانات وحفظ العمليات في ملف `locations-detected.json`.
- **`frontend/`**: تطبيق الـ React المبني بـ Vite مع واجهة مستخدم مظلمة ومؤثرات زجاجية ممتازة.

---

## 💡 معلومات هامة
* للحصول على موقعك بدقة 100% (مثال: **قنا**)، تأكد من الموافقة على طلب المتصفح لمشاركة الموقع الجغرافي (Allow Location Access).
* إذا تم رفض إذن الموقع، سيقوم النظام تلقائياً بتحديد موقع الـ IP التقريبي الذي توفره شركة الاتصالات الخاصة بك (والذي قد يشير إلى الجيزة أو القاهرة).
